Definition <form> :

  • A form is a section of an HTML document specifically designed to gather user input.
  • It consists of various elements like text fields, radio buttons, checkboxes, dropdown menus, and more.
  • Users interact with these elements to provide information, and the form collects and submits it for processing.
  • The action attribute specifies where the data to be sent.
  • The method attributes specifies which HTTP method to be used.
